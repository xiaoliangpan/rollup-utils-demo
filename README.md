## 相关命令如下

```sh
# 构建代码
pnpm build

# 交互式提交git代码
pnpm commit

# eslint代码风格和语法检查
pnpm lint

# eslint代码风格和语法检查 并修复
pnpm lint:fix

# TS检查类型 先执行 pnpm lint
pnpm tsc

# 交互式更新版本号
pnpm release:version

# 生成changelog日志

pnpm release:changelog

# 发包到npm市场
pnpm release
```
